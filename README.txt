In order to run this project you need to have access to XCODE 6 and above. Go inside the file UKMaAPP and then click on the project file .xcodeproj. After clicking the project file Xcode should open the project. To run on the simulator you can click play button.Note that a lot of the interactions between class files are done using interface builder and storyboards. Another thing to note is that the database used is sqlite and is the file with .sqlite. In order to edit database I used FireFox and sqlite manager add on. This made it simple.
